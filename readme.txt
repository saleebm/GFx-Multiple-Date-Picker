An easier way to manage calendar limitations on multiple forms. Requires Gravity Forms and Gravity Perks.

== Description ==

This plugin lets you create and manage calendar limitations of Gravity Forms Calendar fields from one central location. It extends the [Gravity Perks Limit Dates extension](https://gravitywiz.com/documentation/gravity-forms-limit-dates/).

## Requirements

Needs Gravity Forms and Gravity Perks, along with the Gravity Perks limit dates perk.

## Set up

First, just go to the plugin settings under the Gravity Forms tab in the WordPress dashboard. If it is your first time activating this app, you can either click the button to initialize the calendars or create a new calendar context.

A calendar context is a group of form fields to block using one view. For example, you could block various fields from multiple calendars in one context.

In addition, there is an "All" context already defined. In this context tab, every form field selected in other contexts will show up under all.
